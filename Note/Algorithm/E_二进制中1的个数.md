## 题目

**[二进制中1的个数](https://www.acwing.com/problem/content/803/)**

## 解析

**Category:** 位运算

## 代码

```c++
#include <bits/stdc++.h>
using namespace std;

int lowBit(int x) {
    return x & -x;
}

int main() {
    int n;
    scanf("%d", &n);
    while (n -- ) {
        int x;
        scanf("%d", &x);
        int res = 0;
        while(x) x -= lowBit(x), res ++;
        printf("%d ", res);
    }
    return 0;
}
```
